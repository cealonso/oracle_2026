Realizar una simulación del impacto de un incremento salarial del 10% aplicado a los empleados cuya remuneración actual sea inferior a USD 7.000.

```
SELECT employee_id,
       first_name,
       last_name,
       salary               AS "Salario Actual",
       salary+(0.10*salary) AS "Simulación 10%"
FROM   employees
WHERE  salary <= 7000
ORDER BY salary ASC;

```

Realizar una actualización efectiva del salario de aquellos empleados cuya remuneración sea inferior a USD 7.000.

```
UPDATE employees
SET    salary = salary + (0.10 * salary)
WHERE  salary <= 7000;
COMMIT;

```

Generar un listado de empleados con el siguiente formato: El apellido alineado a la izquierda con puntos hasta 25 caracteres, seguido del salario alineado a la derecha con puntos hasta 10 caracteres,


```
SELECT RPAD(last_name, 25, '.') || LPAD(salary, 10, '.') as "Listado de Salarios" FROM employees ORDER BY salary ASC;

```
