Listar los empleados cuyo salario sea mayor al promedio salarial del departamento donde trabaja. (Usar Subquerys)

```
SELECT
	e.department_id as "Cod. de Departamento",
	e.employee_id as "legajo", 
	e.first_name ||' ' || e.last_name as "nombre completo", 
	e.salary as "salario", 
	round((SELECT AVG(salary) FROM employees WHERE department_id=e.department_id),2) 
	as "Promedio del departamento" 
FROM employees e 
WHERE e.salary>(SELECT AVG(salary) FROM employees WHERE department_id=e.department_id) ORDER BY e.department_id

```

**PROBLEMA:** Consulta muy compleja, se repite una parte del código, es lenta.

El mismo problema resuelto a través de CTE:

```
-- PASO 1: Creamos el CTE con los promedios por departamento
WITH promedios_por_depto AS (
    SELECT 
        department_id,
        ROUND(AVG(salary), 2) AS promedio_depto
    FROM employees
    GROUP BY department_id
)

-- PASO 2: Usamos el CTE como si fuera una tabla normal
SELECT 
    e.department_id         AS "Cod. de Departamento",
    e.employee_id           AS "Legajo",
    e.first_name || ' ' || e.last_name  AS "Nombre Completo",
    e.salary                AS "Salario",
    p.promedio_depto        AS "Promedio del Departamento"
FROM employees e
JOIN promedios_por_depto p 
    ON e.department_id = p.department_id
WHERE e.salary > p.promedio_depto
ORDER BY e.department_id;

```
