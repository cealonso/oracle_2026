Mostrar el salario máximo de cada departamento junto al legajo del empleado que cobra dicho salario:

**OTRA SOLUCIÓN:** 

```
SELECT DISTINCT
    department_id,
    FIRST_VALUE(employee_id) OVER (PARTITION BY department_id ORDER BY salary DESC) AS 
    highest-paid employee,
    FIRST_VALUE(salary) OVER (PARTITION BY department_id ORDER BY salary DESC) AS  
    salario_max
FROM employees
WHERE department_id IS NOT NULL
ORDER BY department_id;
```

OJO! con esta consulta en caso de empate para el primer puesto solo devuelve el primero empleado que encuentre.

**OTRA SOLUCIÓN:**

Usando QUALIFY de Oracle 26ai:

```
SELECT 
     employee_id, 
     department_id, 
     salary, 
     RANK() OVER (PARTITION BY department_id ORDER BY salary DESC) AS rnk FROM employees 
QUALIFY rnk = 1;
```

¿Cuál es el empleado más antiguo en cada departamento?

```
SELECT 
    employee_id,
    first_name,
    department_id,
    hire_date,
    LAG(hire_date)
    OVER(PARTITION BY department_id ORDER BY hire_date) AS prev_hiredate
FROM employees;
```

Obtener, por cada empleado, el nombre y apellido del siguiente empleado contratado dentro del mismo departamento:

```
SELECT
    employee_id,
    first_name,
    last_name,
    hire_date,
    department_id,
    LEAD(first_name || ' ' || last_name) OVER (PARTITION BY department_id      
    ORDER BY hire_date) AS "Siguiente Contratado"
FROM employees;

```

Usar percentiles para medir el salario de cada empleado dentro de cada departamento:

```
SELECT
    employee_id, 
    department_id, 
    salary, 
    PERCENT_RANK() OVER (PARTITION BY department_id ORDER BY salary) * 100 as 
    Percent 
FROM employees;
```

Usar la mediana para comparar los salarios por debajo del 50% de cada uno de los departamentos:

```
SELECT 
     employee_id, 
     first_name, 
     department_id, 
     salary, ROUND(PERCENT_RANK() OVER (PARTITION BY department_id ORDER BY 
     salary) * 100, 2) AS percentil 
FROM employees 
QUALIFY percentil < 50;
```

Pivotear filas en una única columna concatenando los valores con el separador coma (,):

```
SELECT department_id, 
       LISTAGG(first_name || ' ' || last_name, ', ') WITHIN GROUP(
       ORDER BY last_name, first_name ) employees
FROM employees 
GROUP BY department_id 
ORDER BY department_id;
```

Pasar esta ultima instrucción ejecutada a una vista SQL:

```
CREATE OR REPLACE VIEW employee_details AS
SELECT department_id, 
       LISTAGG(first_name || ' ' || last_name, ', ') WITHIN GROUP(
       ORDER BY last_name, first_name ) employees
FROM employees 
GROUP BY department_id 
ORDER BY department_id;
```

Ejecutar la vista recién creada:

```
SELECT * FROM employee_details;
```
