Generar una tabla denominada Courses para almacenar todos los cursos que se brindarán en las capacitaciones:

```
CREATE TABLE courses (
    course_id NUMBER GENERATED BY DEFAULT ON NULL AS IDENTITY,
    course_name VARCHAR2(100), 
    start_date DATE NOT NULL, 
    end_date DATE,
    instructor_id NUMBER,
    CONSTRAINT fk_instructor_id FOREIGN KEY (instructor_id) 
    REFERENCES instructors (instructor_id), 
    CONSTRAINT pk_course_id PRIMARY KEY (course_id),
    CONSTRAINT chk_dates CHECK (start_date <= end_date)
);
```

Se necesita modificar la columna course_name con la restricción NOT NULL:

```
ALTER TABLE courses MODIFY course_name varchar2(100) NOT NULL;
```

Se necesita volver a modificar la columna course_name modificando la columna a un máximo de 50 caracteres:

```
ALTER TABLE courses MODIFY course_name varchar2(50);
```

Se necesita modificar la columna start_date para que tenga la fecha actual por defecto cuando  se incorpora un nuevo registro:

```
ALTER TABLE courses MODIFY start_date date DEFAULT sysdate;
```

Se necesita incorporar una nueva columna denominada difficultylevel a la tabla courses para ingresar el nivel de dificultad del curso:

```
ALTER TABLE courses ADD difficultylevel varchar2(20);
```

Modificar la tabla courses para que la columna difficultylevel admita solo los valores low,medium y high.

```
ALTER TABLE courses ADD CONSTRAINT chk_difficultylevel CHECK (difficultylevel IN ('low','medium','high'));
```


Crear un índice sobre la columna course_name de la tabla courses

```
CREATE INDEX idx_course_name ON courses (course_name);
```
