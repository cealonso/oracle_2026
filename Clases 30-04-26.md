Necesito obtener los empleados cuyo salario sea mayor al promedio salarial de la organización.

```
SELECT
    employee_id,
    first_name,
    last_name,
    department_id,
    salary
FROM
    hr.employees
WHERE
    salary > (
        SELECT AVG(salary)
        FROM   hr.employees
        WHERE  department_id = 50
    )
ORDER BY
    salary DESC; 

```

Obtener los 5 empleados con los salarios más altos entre aquellos que superan el salario promedio del departamento 10.

```

SELECT
    employee_id,
    first_name,
    last_name,
    department_id,
    salary
FROM
    employees
WHERE
    salary > (
        SELECT AVG(salary)
        FROM   employees
        WHERE  department_id = 10
    )
ORDER BY
    salary DESC
FETCH FIRST 5 ROWS ONLY;

```
