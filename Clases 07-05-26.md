Mostrar en formato JSON los siguientes datos (employee_id, first_name, last_name, hire_date y salary) del empleados cuyo id es 101.

```
SELECT JSON_OBJECT(
    'employeeId' VALUE employee_id,
    'firstName' VALUE first_name,
    'lastName' VALUE last_name,
    'email' VALUE email,
    'hireDate' VALUE hire_date,
    'salary' VALUE salary
) AS employee_json
FROM employees WHERE employee_id = 101;

```

Crear la tabla instructors con los siguientes campos:

```
CREATE TABLE instructors (
   instructor_id NUMBER GENERATED BY DEFAULT ON NULL AS IDENTITY PRIMARY KEY,
   data JSON
);

```

Mostrar la estructura de la tabla instructors:

```
desc instructors; 

```

```

Listar la información recién ingresada:

```

```

Listar la información recién ingresada en formato mas accesible:

```
SELECT JSON_SERIALIZE(i.data PRETTY) FROM instructors i;
