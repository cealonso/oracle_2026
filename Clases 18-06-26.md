Se necesita realizar un análisis de los empleados cuyo departamento es el numero 50 para determinar un posible aumento salarial.  El aumento seria del 10% pero el salario propuesto no debería sobrepasar el techo máximo de ese puesto.

1) Crear la tabla temporal gtt_employees_salary_data:

```
CREATE GLOBAL TEMPORARY TABLE gtt_employees_salary_data (
     employee_id NUMBER, 
     full_name VARCHAR2(100), 
     current_salary NUMBER, 
     proposed_salary NUMBER
) ON COMMIT PRESERVE ROWS;
```

2) Seleccionar los empleados que trabajan en dicho de departamento y cargarlo en la nueva tabla:

```
INSERT INTO gtt_employees_salary_data
    SELECT employee_id, first_name || ',' || last_name, salary, salary+(salary*0.10)
    FROM employees
    WHERE department_id = 50;
```

3) Verificar la carga de los empleados en dicha tabla:


```
SELECT  * FROM gtt_employees_salary_data ORDER BY employee_id;
```

4)  Verificar el techo salarial de cada empleado:

```
SELECT e.employee_id,e.job_id,e.department_id,j.max_salary FROM employees e JOIN jobs j
ON e.job_id=j.job_id WHERE department_id=50 order by employee_id;

```

5) Validar el aumento del empleado teniendo en cuenta el techo salarial:


```
SELECT gtt.employee_id,gtt.current_salary,gtt.proposed_salary, j.max_salary, 
    CASE 
        WHEN gtt.proposed_salary>j.max_salary THEN 'Supera el máximo salarial'
        ELSE 'Empleado con posibilidad de aumento'
    END AS aumento_salarial
FROM gtt_employees_salary_data gtt JOIN employees e 
ON gtt.employee_id=e.employee_id 
JOIN jobs j ON e.job_id=j.job_id
ORDER BY e.employee_id;

```
