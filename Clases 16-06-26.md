Devolver un listado de los empleados que **ganan más del 70%** del salario máximo permitido de su puesto laboral. Mostrar Los resultados  ordenados de mayor a menor salario:

**UNA SOLUCIÓN: USANDO SUBCONSULTAS**

```
SELECT 
    j.job_title,
    e.first_name || ' ' || e.last_name AS full_name,
    e.salary AS actual_salary,
    j.max_salary AS top_salary
FROM employees e
INNER JOIN jobs j ON e.job_id = j.job_id
WHERE e.salary > (
    -- Subconsulta que calcula el 70% del máximo para EL puesto de esa fila
    SELECT j2.max_salary * 0.70 
    FROM jobs j2 
    WHERE j2.job_id = e.job_id
)
ORDER BY e.salary DESC;
```

**OTRA SOLUCIÓN: USANDO CTEs.**


```
WITH job_limit AS (
    SELECT job_id, job_title, max_salary, max_salary * 0.70 AS critical_limit 
    FROM jobs
)
SELECT 
    jl.job_title,
    e.first_name || ' ' || e.last_name AS full_name,
    e.salary,
    jl.max_salary   
FROM employees e 
JOIN job_limit jl ON e.job_id = jl.job_id
WHERE e.salary > jl.critical_limit
ORDER BY e.salary DESC;

```

Agregar nuevos requerimientos a la ultima consulta: 

-  Diferencia entre el máximo salario que puede obtener de su puesto con su salario real.
- Fecha de contratación.
- Nombre del departamento donde trabaja.
- Antigüedad 


```
WITH job_limit AS (
    SELECT job_id, job_title, max_salary, max_salary * 0.70 AS critical_limit 
    FROM jobs
)
SELECT 
    jl.job_title,
    e.first_name || ' ' || e.last_name AS full_name,
    e.salary,
    jl.max_salary,
    jl.max_salary - e.salary AS difference,
    d.department_name,
    e.hire_date,
    round(months_between(SYSDATE,hire_date)/12,1) AS years_of_service
FROM employees e 
JOIN job_limit jl ON e.job_id = jl.job_id
JOIN departments d ON d.department_id = e.department_id
WHERE e.salary > jl.critical_limit
ORDER BY e.salary DESC;


```
